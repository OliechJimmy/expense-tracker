## Expense Tracker & Budget Analyser

A personal finance data science project that analyses 1000 expense 
transactions across 33 months to uncover spending patterns and 
compare against budget targets.

# Key Questions Answered
- Which category do I spend the most on?
- How does my spending change month to month?
- Which payment mode do I use most?
- Am I over or under budget in each category?

## Key Findings
- Food is the highest spending category
- August is the peak spending month
- Health, Shopping and Transport exceed monthly budget targets
- Spending is evenly distributed across cash, card and online payments

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## How to Run
1. Clone this repository
2. Install dependencies: pip install pandas matplotlib
3. Open expense_analysis.ipynb in Jupyter
4. Run all cells
